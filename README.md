# Python-Password-Validation-Program
*Description

This project is a simple Python program that validates a user's password based on basic security criteria.

The program asks the user to enter a username and password, then checks whether the password meets several requirements. The user has up to three attempts to create a valid password.

*Password requirements
Must be longer than 8 characters
Must contain at least one lowercase letter
Must contain at least one uppercase letter
Must contain at least one number
Must not be the same as the username

*Features
User input validation
Character type checking (lowercase, uppercase, digits)
Limited number of attempts (3 tries)
Clear feedback messages for the user

*Technologies Used
Python 3
Built-in string methods (islower(), isupper(), isdigit())

*Example Output
Welcome to checking your password!

Username: alice
Password: Alice1234

Hi alice, your password has been confirmed
